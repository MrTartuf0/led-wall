## Come configurare l'app led-wall

Clonare questa repository 

```bash
git clone https://github.com/MrTartuf0/led-wall.git
cd led-wall
```

Installare le dipendenze necessarie

```bash
npm i  
```

Eseguire la web-app in modalità development

```bash
npm run dev  
```
